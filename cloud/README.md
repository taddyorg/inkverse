# Cloudflare Edge Services

This is the main repository for Inkverse's Cloudflare Edge Services.

## Steps to Setup

### 1. Install Cloudflare wrangler

```
brew install wrangler
```

### 2. Login to Cloudflare

```
wrangler login
```

### 3. Create a new worker

```
wrangler dev
```

### 4. Deploy the worker

```
wrangler deploy
```
